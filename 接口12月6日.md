##登录
```
POST /user/auth
```
Parma
```
{
 "username":"sbin",
 "password":"sbin"
}
```
Response
```
{
 "username":"sbin",
 "type":"student"|"teacher"|"admin",
 "avatar":"image url",
 "gender":"male"|"female",
 "email":"xxx@xxx.com",

 //student attribute
 "gitId":int,
 "number":"141250123",


 //teacher arrribute
 "authority":"true"|"false",

}
```
