# 接口12.6

---

> note:
>
> 所有url都有/api前缀
>
> 除/user/auth之外,所有接口到要通过http基本认证

## 登录

```
POST /user/auth
```

Parma

```
{
 "username":"sbin",
 "password":"sbin"
}
```

Response

```
{
 "username":"sbin",
 "type":"student"|"teacher"|"admin",
 "avatar":"image url",
 "gender":"male"|"female",
 "email":"xxx@xxx.com",

 //student attribute
 "gitId":int,
 "number":"141250123",


 //teacher arrribute
 "authority":"true"|"false",

}
```

## 查看学生/教师课程

```
GET /user/{userId}/course
```

Response

```
[
   {
    "id":123
    "name":"14软工II2班",
    "description":"string"
   },
]
```

## 查看学生考试

```
GET /user/{userId}/course/{courseId}/exam
```
